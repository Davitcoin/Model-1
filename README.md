# Simple Connection to test Gemini Q&A

## Before you start

1. Make sure you have your Gemini API and add it in an **.env** file 

    `Key="YOUR_GEMINI_API_KEY_HERE`


## To run: 
2. Make sure you have installed all the required dependencies, If not run: 
    `npm install`

3. Run:

    `node index.js`
